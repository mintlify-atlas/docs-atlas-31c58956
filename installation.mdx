## Install the package

You can install Nanolytics using npm, yarn, or pnpm:

<CodeGroup>

```bash npm
npm install nanolytics
```

```bash yarn
yarn add nanolytics
```

```bash pnpm
pnpm i nanolytics
```

</CodeGroup>

## Requirements

<Info>
  Nanolytics requires **Node.js >= 20.0.0** to run.
</Info>

## What's included

When you install Nanolytics, you get:

- The main library bundle (`dist/main.js`)
- TypeScript type definitions (`dist/*.d.ts`)
- Source maps for debugging (`dist/main.js.map`)
